VOICI TOUTES LES COMMANDES DISPONIBLES SUR LE SERVEUR.

/mp [destinataire] [message]  
ENVOI UN MESSAGE A UN DESTINAIRE
SI VOUS ECRIVEZ ALL POUR LE DESTINATAIRE CELA ENVERRA UN MESSAGE À TOUTES LES PERSONNES 
PRESENTES SUR LE SERVEUR.
ATTENTION À BIEN RESPECTER L'ORTHOGRAPHE DU DESTINATAIRE ET EVITEZ LES INSULTES.

/file [nomFichier]  
VOUS RECUPEREZ LE FICHIER VOULU.
ATTENTION À BIEN RESPECTER L'ORTHOGRAPHE DU NOM DE FICHIER.

/getFile
LE SERVEUR VOUS ENVOI LA LISTE DE TOUT LES FICHIERS PRESENTS SUR LE SERVEUR.

/getFile [nomfichier]
VOUS RECUPEREZ LE FICHIER VOULU.
ATTENTION À BIEN RESPECTER L'ORTHOGRAPHE DU NOM DE FICHIER.

/salonList 
LE SERVEUR VOUS ENVOI LA LISTE DE TOUT LES SALONS PRESENTS SUR LE SERVEUR.

/salon [message]
VOUS ENVOYEZ UN MESSAGE A TOUT LES CLIENTS CONNECTÉS SUR VOTRE SALON.

/newSalon [nomSalon] [capacite] [description]
VOUS AJOUTEZ UN SALON VOULU. 
ATTENTION A BIEN RESPECTER LE NOMBRE D'ARGUMENTS À RENTRER EN PARAMETRE. 
LE NOM DU SALON S'ECRIT SANS ESPACE.

/modifChannel [nomSalon][nouveauNomSalon][nouvelleCapacite] [nouvelleDescription] 
VOUS MODIFIEZ LE SALON VOULU. 
ATTENTION À BIEN RESPECTER L'ORTHOGRAPHE DU SALON ET DE VERIFIER 
QUE VOUS AVEZ LES DROITS SINON LE SALON NE SE MODIFIERA PAS.

/delChannel [nomSalon]
VOUS SUPPRIMEZ LE SALON VOULU. 
ATTENTION À BIEN RESPECTER L'ORTHOGRAPHE DU SALON ET DE VERIFIER 
QUE VOUS AVEZ LES DROITS SINON LE SALON NE SE SUPPRIMERA PAS.

/joinSalon [nomSalon]
VOUS REJOIGNEZ LE SALON VOULU. 
ATTENTION À BIEN RESPECTER L'ORTHOGRAPHE DU SALON.

/fin 
VOUS QUITTEZ LE SERVEUR.
